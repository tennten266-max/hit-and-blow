# Hit and Blow (Numer0n Style) in C

C言語で実装された、4桁の数字を当てる「ヒット＆ブロー（数当てゲーム）」です。
プログラムがランダムに生成した4桁の数字を、ヒント（HITとBLOW）を頼りに解き明かします。

## 🎮 ゲームルール

1. **HIT (ヒット)**: 数字も位置も合っている場合
2. **BLOW (ブロー)**: 数字は合っているが、位置が違う場合

* 桁数は **4桁** です。
* 先頭の数字に `0` は選ばれません（必ず1000〜9999の範囲になります）。
* 正解の数字には、同じ数字が重複して含まれる場合があります。
* 途中で `0` を入力すると、降参（ギブアップ）して正解を表示し、ゲームを終了します。

## 🛠️ 動作環境・コンパイル方法

標準的なC言語の環境であれば動作します。

gcc -o hit_and_blow main.c
